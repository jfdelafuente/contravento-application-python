1. **Go to trip detail page** (from "Mis Viajes")

### Unpublishing a Trip

You can make a published trip private again:

1. **Go to trip detail page**
2. **Click "Despublicar"** button
3. **Confirm** the action
4. **Trip returns to draft status** (private)

- Trip becomes private (only you can see it)
- Existing comments and likes are preserved
- Not visible in public feed
- Can be re-published anytime

## Troubleshooting

### "Cannot publish trip" Error

**Possible causes**:
1. **Description too short**: Must be â‰¥50 characters
   - Solution: Add more detail to description

2. **Invalid dates**: End date before start date
   - Solution: Check dates are in correct order

3. **Email not verified**: Account not verified
   - Solution: Check email and verify account

### Photos Not Uploading

**Possible causes**:
1. **File too large**: >10MB per photo
   - Solution: Resize photos before upload

2. **Wrong format**: Not JPG/PNG/WebP
   - Solution: Convert to supported format

3. **Network error**: Slow connection
   - Solution: Wait and retry

4. **Max photos reached**: Already have 20 photos
   - Solution: Delete some photos first

### Trip Not Appearing in Feed

**Check**:
1. Is trip **published**? (Not draft)
2. Is account **verified**?
3. Did you **refresh** the feed page?
4. Are **filters** applied? (remove filters)
